# Sistema de Agendamento de Consultas (SAC)

Projeto desenvolvido para a disciplina **Prática Profissional em Análise e Desenvolvimento de Sistemas (ADS)** – Mackenzie.  
O sistema permite agendar, listar e cancelar consultas de forma simples, via navegador.

---

## 🩺 Funcionalidades principais

- Login de usuário (admin)
- Agendamento de consultas
- Listagem de consultas cadastradas
- Cancelamento de consultas
- Mensagens de confirmação na interface

> Observação: algumas funcionalidades mais avançadas (confirmação de presença, relatórios, notificações externas etc.) estão previstas para iterações futuras.

---

## 🛠 Tecnologias utilizadas

- **Node.js** + **Express**
- **SQLite3** (banco de dados)
- **HTML5, CSS3** e **Bootstrap**
- Deploy em **Render.com**

---

## ✅ Pré-requisitos

Para executar localmente, é necessário:

- **Node.js** 18 ou superior instalado
- **Git** instalado
- Acesso à internet para instalar dependências via `npm`
